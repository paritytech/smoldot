Expand description

Configuration to pass when the actual block authoring is started.

Fields§

§block_number_bytes: usize

Number of bytes used to encode block numbers in the header.

§parent_hash: &'a [u8; 32]

Hash of the parent of the block to generate.

Used to populate the header of the new block.

§parent_number: u64

Height of the parent of the block to generate.

Used to populate the header of the new block.

§now_from_unix_epoch: Duration

Time elapsed since the Unix Epoch (i.e. 00:00:00 UTC on 1 January 1970), ignoring leap seconds.

§parent_runtime: HostVmPrototype

Runtime used to check the new block. Must be built using the Wasm code found at the :code key of the parent block storage.

§top_trie_root_calculation_cache: Option<CalculationCache>

Optional cache corresponding to the storage trie root hash calculation coming from the parent block verification.

§block_body_capacity: usize

Capacity to reserve for the number of extrinsics. Should be higher than the approximate number of extrinsics that are going to be applied.
