Substrate/Polkadot-specific protocols on top of libp2p.

Libp2p provides a framework on top of which higher-level, domain-specific protocols can be used. This module contains the domain-specific protocols specific to Substrate and Polkadot.

This module only provides the tools to encode/decode messages.
